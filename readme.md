# 🍬 Jodhpur Sweet Store Management System

## 📌 Project Overview
Jodhpur Sweet Store Management System is a **Windows Desktop Application** developed in **C# .NET (WinForms)** to manage day-to-day operations of a sweet shop.

This software helps in handling **billing, customer records, sales tracking, and reports** efficiently.

---

## 🚀 Features

- 🧾 **Billing System**
  - Generate invoices quickly
  - Manage daily sales

- 👥 **Customer Management**
  - Store customer details
  - Track purchase history

- 💰 **Payment Tracking**
  - Manage receipts and dues
  - Track pending payments

- 📊 **Reports**
  - Sales reports
  - Payment reports
  - Customer reports

- 🗂️ **Database Management**
  - Secure data storage using SQL Server

---

## 🛠️ Technologies Used

- 💻 **Frontend:** C# .NET (Windows Forms)
- 🗄️ **Database:** SQL Server 2012
- 📈 **Reporting:** SAP Crystal Reports
- 🧰 **IDE:** Visual Studio 2022

---

## ⚙️ System Requirements

- Windows OS (Windows 10/11 recommended)
- Visual Studio 2019/2022
- SQL Server 2012 or later
- .NET Framework (as required by project)
